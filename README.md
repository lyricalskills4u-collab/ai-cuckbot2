# Android AI Chatbot

An Android application featuring an AI-powered chatbot with character creation, messaging, photo gallery, and settings management.

## Features

### 🎭 Meetup (Character Design)
- Create and design custom characters
- Customize appearance: hair color, skin tone, body type
- Set physical attributes: breast/ass size (female), cock size (male)
- Design interactions and chat behaviors
- Support for NSFW content

### 💬 Messages
- Real-time messaging with AI characters
- Send and receive text messages
- Share pictures and videos
- Chat history management

### 🖼️ Photos
- Gallery view of all media from chats
- View saved pictures and videos
- Organize and manage content

### ⚙️ Settings
- Save and load chat sessions
- App preferences
- Close application
- Data management

### 📹 Video Chat (Future Update)
- Placeholder for video chat functionality

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Android Jetpack (Compose/Traditional)
- **Database**: Room Database
- **Architecture**: MVVM + Repository Pattern
- **Networking**: Retrofit (for AI API calls)
- **Media**: MediaStore, File Management

## Getting Started

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle files
4. Run on emulator or device
